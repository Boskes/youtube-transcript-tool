# 🎬 YouTube Transcript Tool

Een Nederlandstalige webapplicatie om YouTube-transcripts op te halen en te bekijken, met duidelijk onderscheid tussen verschillende soorten tekst.

## Functies

- 📝 **Handmatig transcript** — Tekst rechtstreeks uit ingebedde ondertiteling (blauw)
- 🤖 **Auto-gegenereerd (ASR)** — Door YouTube automatisch herkende spraak (amber/geel)
- 🎵 **Annotaties** — Muziek, geluiden of beschrijvingen tussen haakjes (groen)
- ⏱ Tijdcodes per fragment
- 🔍 Zoekfunctie in het transcript
- 📋 Kopieerknop voor het volledige transcript (met of zonder tijdcodes)
- 🌐 Taalvoorkeur instelbaar

## Installatie & Gebruik

```bash
cd youtube-transcript-tool
npm install
npm start
```

Open vervolgens: **http://localhost:3000**

## Ondersteunde URL-formaten

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/shorts/VIDEO_ID`
- `https://www.youtube.com/live/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`
- Bare video ID (11 tekens)

## API

### `GET /api/transcript?url=<youtube-url>&lang=<taalcode>`

Geeft JSON terug:
```json
{
  "videoId": "...",
  "language": "nl",
  "isAutoGenerated": false,
  "segments": [
    { "index": 0, "text": "...", "timestamp": "0:05", "type": "transcript" }
  ],
  "fullText": "...",
  "plainText": "...",
  "stats": { "total": 100, "transcript": 90, "auto": 5, "annotation": 5 }
}
```

**Segment types:**
- `transcript` — handmatige ondertiteling
- `auto` — automatisch gegenereerd (ASR)
- `annotation` — muziek/geluid annotaties

## Technologie

- **Backend:** Node.js + Express (ESM)
- **Frontend:** Vanilla HTML/CSS/JS
- **YouTube API:** YouTube InnerTube API (geen API-sleutel nodig)
